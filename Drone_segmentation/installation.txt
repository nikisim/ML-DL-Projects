Всем привет!
Для выполнения тестового задания нам понадобится Python и некоторое количество библиотек.
В идеале нам потребуется также GPU для его выполнения. 
Можно все разрабатывать локально, а потом перейти, например, на Kaggle Kernel и запустить все там уже на GPU.

Для локальной разработки если работаете с Python впервые, то проще всего установить анаконду:
https://docs.anaconda.com/anaconda/install/index.html
Если уже используете Python, то, скорее всего, сами знаете как его установить.

Далее нам потребуются следующие пакеты:

pip install Pillow
pip install numpy
pip install opencv-python
pip install torch
pip install segmentation-models-pytorch


Для разработки на Kaggle Kernel необходимо зарегистрироваться на этом сайте 
https://www.kaggle.com

Далее пройти сюда 
https://www.kaggle.com/code
Выбрать там New Notebook

Далее можно заниматься разработкой. Когда решите использовать GPU можно нажать на три точки (бутерброд) -> Accelerator -> GPU.
